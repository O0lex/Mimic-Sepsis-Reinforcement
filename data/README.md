# Data Folder Notes

This folder is for local datasets only.

- Public demo data is allowed in git:
	- `mimic-iv-clinical-database-demo-on-fhir-2.0/`
- Do not commit credentialed/private MIMIC-IV files or non-demo extracts.
- Keep only lightweight metadata or instructions in git.
- The main pipeline expects local data paths and writes processed outputs under `outputs/data/`.
